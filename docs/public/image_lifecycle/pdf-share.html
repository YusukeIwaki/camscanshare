Rules

  • 未 apply の working preview は共有結果に反映しない。使うのは保存済み metadata のみ。
  • preview cache が壊れていても、source と metadata が正しければ PDF は正しく作れる。
  • temp PDF は preview refresh と独立に再生成できる。
