CamScanShare Asset Studio

HTML + CSS からアイコンとフィルタサムネイルを生成するためのベース。
`?asset=` を付けるとキャプチャ用の単体表示になります。

Android App Icon

紙面、スキャン枠、カメラレンズを一体化した launcher アイコン。adaptive icon 系もこの見た目を基準にする。

iOS App Icon

丸いオーブ背景を外し、メインモチーフを拡大した iOS 用アイコン。

Original

素の紙面。色もコントラストも控えめに見せる。

Sharpen

線を濃く、輪郭を締めて「くっきり」を一目で出す。

BW

完全なモノクロ。紙とインクだけに整理した見た目。

Magic

紙面を明るく整えつつ、少しだけ色を残す自動補正の見た目。

Whiteboard

白板を白く、マーカー色を残す専用フィルタをそのまま表現。

Vivid

図表や色付き文書向けに、色面を強く目立たせる。
